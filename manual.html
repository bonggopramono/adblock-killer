Untuk menerapkan adblock killer pada blog, dilakukan dengan: [1] Meng-copy paste isi dari file adblock-killer.css ke template blog dan letakkan diantara atau [2] Letakkan kode berikut /* Adblock Killer */ diatas [3] Ganti ALAMAT_CDN_HOST_ANDA.js dengan alamat host CDN file adblock-killer.js tempat menyimpan file-nya
